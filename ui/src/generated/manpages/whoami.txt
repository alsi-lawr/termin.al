NAME
     whoami - Show the current development capability.

SYNOPSIS
     whoami

DESCRIPTION
     Prints the fixed capability identity used by the public browser demo.

OUTPUT
     The  exact  output  is anonymous.  It describes the current unauthenticated
     development capability, not an operating-system account, GitHub identity or
     browser profile.

FAILURES
     Any argument is rejected.

NOTES
     The value is not read from credentials or browser storage and is  unchanged
     by the unavailable login and logout commands.

EXAMPLES
     $ whoami

SEE ALSO
     login(1), logout(1), stats(1)
