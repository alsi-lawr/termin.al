NAME
     whoami - Show the current development capability.

SYNOPSIS
     whoami

DESCRIPTION
     Refreshes  and  prints  the current anonymous, GitHub-viewer, owner and CV-
     viewer capability state. GitHub states include the authenticated login.

FAILURES
     Arguments are rejected. When authentication is unavailable, public browsing
     continues and the command reports anonymous authentication-unavailable sta‐
     tus.

NOTES
     Identity is shown only in the terminal session and is not sent to site sta‐
     tistics.

EXAMPLES
     $ whoami

SEE ALSO
     login(1), logout(1), cv(1), stats(1)
