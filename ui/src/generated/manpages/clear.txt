NAME
     clear - Clear terminal scrollback.

SYNOPSIS
     clear

DESCRIPTION
     Removes  the  visible  terminal  transcript from the current shell pane. It
     does not close an inline viewer because clear runs only from the shell pre‐
     sentation.

OUTPUT
     No text is printed. The clear-scrollback effect is applied after  the  com‐
     mand succeeds.

FAILURES
     Any argument is rejected and leaves the scrollback unchanged.

NOTES
     Command  history,  the  current virtual directory, other panes, pane layout
     and browser theme preference are preserved. The cleared transcript  is  not
     recoverable  through  history  because  history stores command sources, not
     prior output.

EXAMPLES
     $ clear

SEE ALSO
     history(1), pane(1), pwd(1)
