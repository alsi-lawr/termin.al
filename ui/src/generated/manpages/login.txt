NAME
     login - Authenticate with GitHub.

SYNOPSIS
     login

DESCRIPTION
     Opens  the  GitHub  authentication flow in a popup. Any GitHub identity re‐
     ceives viewer capability; the configured numeric  owner  identity  receives
     owner and CV capabilities.

OUTPUT
     On success, prints the authenticated GitHub login and capability. Authenti‐
     cation failures use one generic diagnostic.

FAILURES
     Arguments  are  rejected.  A blocked popup, invalid or expired correlation,
     unavailable configuration, failed token exchange, or five-second completion
     timeout reports Authentication failed..

NOTES
     Tokens are never placed in terminal output, browser-visible URLs  or  popup
     messages. The offline demo changes only its synthetic in-memory session and
     performs no OAuth or network operation.

EXAMPLES
     $ login

SEE ALSO
     logout(1), whoami(1), cv(1)
