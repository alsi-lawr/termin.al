NAME
     login - Authenticate with GitHub.

SYNOPSIS
     login

DESCRIPTION
     Identifies  the authentication command name, but host authentication is un‐
     available in the current browser demo. No GitHub  or  other  authentication
     flow begins.

FAILURES
     Every invocation returns the stable command-rejected diagnostic:

     Authentication  is unavailable until the host authentication work is imple‐
     mented.

     Arguments do not change this outcome.

NOTES
     No credential, token, cookie, session or browser-storage value is  created.
     This manual makes no promise about a future authentication protocol.

EXAMPLES
     $ login

SEE ALSO
     logout(1), whoami(1), cv(1)
