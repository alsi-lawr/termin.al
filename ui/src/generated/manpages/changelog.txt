NAME
     changelog - Open changelog content.

SYNOPSIS
     changelog

DESCRIPTION
     Opens the fixed virtual document ~/changelog.md as rendered Markdown in the
     current pane's inline viewer.

VIEWER
     Use  j/k  or  arrows  to  move, PageUp/b and PageDown/Space to page, g/G to
     jump, / to search, n/N to move between matches, and q or Escape  to  return
     to the same shell. Mobile controls provide Escape, movement and Prefix.

FAILURES
     Extra  arguments are rejected. Missing content produces the stable content-
     unavailable diagnostic; cancellation opens nothing.

NOTES
     The changelog is a read-only published document.  Returning  preserves  the
     pane's transcript, history and working directory.

EXAMPLES
     $ changelog

SEE ALSO
     now(1), about(1), open(1)
