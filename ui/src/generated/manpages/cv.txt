NAME
     cv - Unlock or open protected CV content.

SYNOPSIS
     cv [--split horizontal|vertical]
     cv lock

DESCRIPTION
     Opens  protected  CV  Markdown  inline or in a split. When shared access is
     locked, requests the viewer key through the non-echoing secret prompt. Own‐
     ers bypass the shared key.

     cv lock removes shared-CV capability  and  discards  open  CV  presentation
     state.

FAILURES
     Positional  keys  and unsupported options are rejected. Invalid, throttled,
     unavailable or missing CV access reports CV access failed..

NOTES
     The viewer key is never accepted as a positional argument, retained in com‐
     mand history, echoed to assistive technology, persisted or logged. CV  con‐
     tent  is  memory-only  and uncounted by statistics. The offline demo serves
     synthetic CV content only.

EXAMPLES
     $ cv
     $ cv --split vertical
     $ cv lock
