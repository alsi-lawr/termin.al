NAME
     cv - Open cv content.

SYNOPSIS
     cv

DESCRIPTION
     Requests  the  fixed  virtual  target ~/cv.md.  In the current browser-demo
     corpus that entry is deliberately locked, so no viewer is opened.

FAILURES
     With no arguments, cv returns  the  stable  diagnostic  Access  is  locked:
     ~/cv.md.   Extra arguments are rejected before target resolution. Cancella‐
     tion opens nothing.

NOTES
     The command does not authenticate, unlock, download or expose  CV  content.
     The virtual corpus remains read-only.

EXAMPLES
     $ cv
