NAME
     less - Open raw virtual document text in a pager.

SYNOPSIS
     less <path>

DESCRIPTION
     Opens  one  virtual document in termin.al's raw pager. The pager keeps GNU-
     less-like presentation separate from the Vim manpage viewer.

EXAMPLES
     $ less notes/sample-note.md

NOTES
     Keys include j/k and the arrow keys, PageUp/b, PageDown/Space, g/G,  Ctrl+f
     and q/Escape. Ctrl+b remains the pane-prefix key.
