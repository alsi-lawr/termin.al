NAME
     theme - List, select, or follow the system terminal theme.

SYNOPSIS
     theme [list|set <name>|system]

DESCRIPTION
     Displays  or changes the terminal colour theme. Explicit selections persist
     in browser storage; system mode follows the active operating-system prefer‐
     ence and clears the stored selection.

OPTIONS
     list   List available theme names.

     set name
            Select one named theme.

     system
            Follow the operating-system colour preference.

EXAMPLES
     $ theme list
     $ theme set gruber-darker
     $ theme system

SEE ALSO
     help(1), man(1)
