NAME
     theme - List, select, or follow the system terminal theme.

SYNOPSIS
     theme [list|set <name>|system]

DESCRIPTION
     Shows or changes the terminal colour theme for the whole browser workspace.
     With no subcommand, theme prints the active name and whether it is explicit
     or derived from the operating-system preference.

SUBCOMMANDS
     list   Print  all six supported names and mark the active one: gruber-dark-
            muted, gruber-dark, gruber-darker, gruber-light-muted,  gruber-light
            and gruber-lighter.

     set name
            Select one exact supported name and store it as the explicit browser
            preference.

     system
            Remove  the stored explicit preference and follow the operating-sys‐
            tem colour scheme. Dark maps to gruber-dark-muted and light maps  to
            gruber-light-muted.

OUTPUT
     Each valid form prints the resulting current theme or available-theme list.

FAILURES
     Unknown  names,  unknown  subcommands and extra or missing subcommand argu‐
     ments are rejected with no preference change.

NOTES
     An explicit selection is stored under this site's browser local storage and
     survives reloads in the same browser profile and origin.  System  mode  re‐
     moves that value. Theme choice is workspace-wide, not pane-local.

EXAMPLES
     $ theme
     $ theme list
     $ theme set gruber-darker
     $ theme system

SEE ALSO
     stats(1), pane(1)
