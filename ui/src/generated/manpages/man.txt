NAME
     man - Show a command manual.

SYNOPSIS
     man <command>
     man -P less|vi <command>
     man --pager=less|vi <command>

DESCRIPTION
     Opens  the  checked  section-1  manual  for  a  registered command. Command
     aliases resolve through the live registry to one canonical manual. The  de‐
     fault  and  explicit  less modes use the same less-style raw pager; vi mode
     opens the complete generated text in the native read-only Vim surface.

OPTIONS
     -P less|vi
            Select the pager. This option must be first, followed by a  separate
            selector and exactly one command name.

     --pager=less|vi
            Select the pager with one option token, followed by exactly one com‐
            mand name.

     The  two  forms  cannot  be combined. The superseded selector vim is unsup‐
     ported, and --pager vi or -Pvi are not accepted spellings.

VIEWER
     Less shows at most 20 source lines per page. j/k or arrows move  one  line;
     PageUp/b  moves  back; PageDown/Space or Ctrl+f moves forward; g/G jumps to
     start/end; q or Escape closes. Its inverse prompt reports  the  page  posi‐
     tion.

     Vi  shows the complete manual with a real caret and NORMAL, visual, command
     and search status. Counts, document and structural  motions,  finds,  local
     marks  and text objects use the shared Vim behavior. Slash searches forward
     and question mark searches backward with native regular expressions; n  re‐
     peats  in  the original direction, N reverses it, both directions wrap, and
     gn or gN selects a search match. Search may also be used as an operator mo‐
     tion in editable Vim buffers.

     The vi manual is read-only. q, :q and :q! close it. Insert, change, delete,
     yank, paste, undo, redo, :w and other write-bearing commands  are  rejected
     without changing the manual or shell. Escape first cancels a prompt, visual
     mode or pending command; a subsequent normal-mode Escape closes.

     Ctrl+b  remains  the  pane prefix and takes precedence over pager b. Mobile
     controls expose Esc, up/down and Prefix. Lines wrap to the pane width with‐
     out horizontal scrolling.

FAILURES
     Missing or extra operands, unsupported option forms, an empty selector, un‐
     supported pagers and unregistered command names are rejected. A  registered
     command whose generated artifact is absent reports no manual entry.

NOTES
     All routes consume the same build-generated UTF-8 artifact. Closing returns
     to  the owning shell with its transcript, history and current directory un‐
     changed; pager and transient vi state are not persisted.

EXAMPLES
     $ man grep
     $ man -P less pane
     $ man --pager=vi theme
