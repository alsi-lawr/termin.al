NAME
     history - Show this shell pane's bounded command history.

SYNOPSIS
     history

DESCRIPTION
     Prints the bounded command history owned by the current shell pane. Entries
     are  numbered  from  1 in retained order and preserve the submitted command
     source, including quoting.

OUTPUT
     The history invocation is already present  in  the  execution  context  and
     therefore appears as the final entry. Number columns widen as required.

FAILURES
     Any argument is rejected.

LIMITS
     Each  shell  pane  retains its latest 100 submitted commands. Older entries
     are discarded when the bound is exceeded.

NOTES
     History is pane-local and survives clear(1), directory  changes  and  pane-
     tree transforms while that shell pane exists. It is discarded when the pane
     closes or the browser page reloads, and it does not include output text.

EXAMPLES
     $ echo one
     $ history

SEE ALSO
     clear(1), pane(1), pwd(1)
