NAME
     logout - End the authenticated session.

SYNOPSIS
     logout

DESCRIPTION
     Ends  the  protected  browser  session and removes owner, GitHub-viewer and
     shared-CV capabilities. Open CV presentation  state  is  discarded.  Editor
     drafts are not changed.

OUTPUT
     Prints Logged out.  on success.

FAILURES
     Arguments  are rejected. Session, origin or request-forgery failures report
     Authentication failed..

NOTES
     The offline demo clears only its synthetic in-memory capability state.

EXAMPLES
     $ logout

SEE ALSO
     login(1), whoami(1), cv(1)
