NAME
     logout - End the authenticated session.

SYNOPSIS
     logout

DESCRIPTION
     Identifies  the session-ending command name, but authentication is unavail‐
     able in the current browser demo. There is no authenticated session for lo‐
     gout to end.

FAILURES
     Every invocation returns the stable command-rejected diagnostic:

     Authentication is unavailable until the host authentication work is  imple‐
     mented.

     Arguments do not change this outcome.

NOTES
     No shell, pane, theme preference, cookie, token or browser-storage value is
     cleared.

EXAMPLES
     $ logout

SEE ALSO
     login(1), whoami(1)
