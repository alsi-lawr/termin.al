NAME
     stats - Show aggregate site statistics.

SYNOPSIS
     stats

DESCRIPTION
     Shows  the  latest  aggregate site statistics available to the session, in‐
     cluding the current connection or freshness status.

EXAMPLES
     $ stats

     help(1), man(1)
