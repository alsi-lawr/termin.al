NAME
     stats - Show aggregate site statistics.

SYNOPSIS
     stats

DESCRIPTION
     Prints  the  workspace's current aggregate statistics snapshot. The command
     reads the latest in-memory stats  state  each  time  it  runs  rather  than
     caching its first output.

OUTPUT
     Available  snapshots  include STATUS, total SESSIONS and VIEWS, the current
     UTC day's counts, and page views sorted by content identifier. Stable  sta‐
     tus  labels are LIVE, RECONNECTING, STALE, NO DATA and UNAVAILABLE. With no
     usable snapshot, only STATISTICS and STATUS UNAVAILABLE are printed.

FAILURES
     Any argument is rejected. Transport failure is represented by  the  current
     stable status rather than a command failure.

NOTES
     In demo mode the UTC date is derived from the fixture snapshot for repeata‐
     bility.  Accepted  document opens may update the session snapshot, but each
     content identifier is submitted at most once per workspace lifetime.  Stats
     command output and workspace state are not stored by this command.

EXAMPLES
     $ stats
