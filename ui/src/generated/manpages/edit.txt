NAME
     edit - Edit published content.

SYNOPSIS
     edit <path>

DESCRIPTION
     Identifies  the  published-content editing command name, but editing is un‐
     available in the current browser demo. No path is resolved and no editor or
     viewer is opened.

FAILURES
     Every invocation returns the stable command-rejected diagnostic:

     Published-content editing is unavailable until  authoring  work  is  imple‐
     mented.

     The same outcome is returned for a missing, valid or extra path argument.

NOTES
     No  virtual document, browser storage or pane state is modified. The public
     corpus is read-only; this manual does not promise an authoring workflow  or
     compatibility surface.

EXAMPLES
     $ edit notes/sample-note.md
     $ edit ~/about.md

SEE ALSO
     open(1), less(1), login(1)
