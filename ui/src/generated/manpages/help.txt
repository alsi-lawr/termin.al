NAME
     help - Show commands grouped by purpose.

SYNOPSIS
     help

DESCRIPTION
     Builds a terminal command index from the live registry and writes it to the
     current  pane's  scrollback.  Commands are grouped as GNU-like, application
     and navigation commands; each row uses the registered summary and  includes
     aliases when present.

OUTPUT
     The  output  is plain terminal text with NAME, SYNOPSIS, COMMANDS, EXAMPLES
     and SEE ALSO headings. It is a compact index, not the full generated manual
     corpus.

FAILURES
     Any argument is rejected.

NOTES
     Use man(1) to open the complete reference for one registered  command.  Un‐
     quoted  and  unescaped redirections support descriptors 0 through 9 with <,
     >, >>, <>, <&, >& and >|.   Redirections  belong  to  each  command  before
     pipelines  and  command lists. Redirected files are shared by all panes and
     persist in browser-local storage; no host filesystem is accessed.

EXAMPLES
     $ help
     $ man open

SEE ALSO
     man(1)
