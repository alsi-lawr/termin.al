NAME
     cat - Print raw virtual document content.

SYNOPSIS
     cat <path> [path ...]

DESCRIPTION
     Reads  one  or  more  files  from termin.al's virtual filesystem and writes
     their raw document text to the current terminal scrollback. Paths  are  re‐
     solved relative to the current pane directory; absolute virtual paths begin
     with ~.  Files are read in argument order without Markdown rendering, head‐
     ings or separators.

OUTPUT
     Each  successful  operand  creates one text output in the same order as its
     argument. An empty document produces an empty text output.

FAILURES
     One to 32 paths are required. Options before -- are  unsupported.  Invalid,
     missing,  directory,  locked or unavailable operands reject the command. If
     any read fails or cancellation occurs, no accumulated partial output is re‐
     turned.

LIMITS
     At most 32 file operands are accepted per invocation.

NOTES
     Published content and browser-persistent redirected files share one virtual
     namespace. cat never reads the browser host or local machine filesystem.

EXAMPLES
     $ cat about.md
     $ cat ~/skills.md ~/tools.md

SEE ALSO
     head(1), tail(1), less(1), pwd(1)
