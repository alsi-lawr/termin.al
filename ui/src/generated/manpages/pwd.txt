NAME
     pwd - Print this shell pane's virtual directory.

SYNOPSIS
     pwd

DESCRIPTION
     Prints  the  canonical virtual working directory owned by the current shell
     pane. Virtual home is written  as  ~  and  descendants  as  paths  such  as
     ~/notes.

OUTPUT
     Exactly one text output is added to the current pane transcript.

FAILURES
     Any argument is rejected.

NOTES
     Each  shell  pane  owns  its  directory  independently.  The value survives
     clear(1) and pane-tree  transforms  while  the  pane  exists,  but  is  not
     browser-persisted across reloads.

EXAMPLES
     $ pwd
     $ cd notes
     $ pwd

SEE ALSO
     cd(1), ls(1), pane(1)
