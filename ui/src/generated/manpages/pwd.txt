NAME
     pwd - Print this shell pane's virtual directory.

SYNOPSIS
     pwd

DESCRIPTION
     Prints  the  virtual working directory of the current shell pane using ter‐
     min.al path notation.

EXAMPLES
     $ pwd

SEE ALSO
     help(1), man(1)
