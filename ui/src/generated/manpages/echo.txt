NAME
     echo - Print arguments as text.

SYNOPSIS
     echo [text ...]

DESCRIPTION
     Writes  its  parsed  arguments  as  one  line  of text, separated by single
     spaces. Shell quoting controls argument boundaries and is not reproduced in
     the output.

OUTPUT
     With no arguments, echo emits an empty text  output.  Option-looking  argu‐
     ments  are printed as ordinary text; echo implements no flags or escape ex‐
     pansion.

EXAMPLES
     $ echo 'hello terminal'
     $ echo -n is text here

NOTES
     Unredirected output is added to the current pane's transcript. Shell  redi‐
     rection  can  write  the  exact text without an added trailing newline to a
     shared browser-persistent virtual file.

SEE ALSO
     clear(1), history(1)
