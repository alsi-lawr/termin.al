NAME
     echo - Print arguments as text.

SYNOPSIS
     echo [text ...]

DESCRIPTION
     Joins  its  arguments  with single spaces and writes the result to terminal
     scrollback.

EXAMPLES
     $ echo 'hello terminal'

SEE ALSO
     help(1), man(1)
