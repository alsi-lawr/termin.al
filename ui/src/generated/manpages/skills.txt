NAME
     skills - Open skills content.

SYNOPSIS
     skills

DESCRIPTION
     Opens  the  fixed  virtual document ~/skills.md as rendered Markdown in the
     current pane's inline viewer.

VIEWER
     Use j/k or arrows to move, PageUp/b and  PageDown/Space  to  page,  g/G  to
     jump,  / to search, n/N for matches, and q or Escape to return. Mobile con‐
     trols provide Escape, movement and Prefix.

FAILURES
     Extra arguments are rejected. Missing content produces the stable  content-
     unavailable diagnostic; cancellation opens nothing.

NOTES
     The  document  is read-only. Returning preserves the shell transcript, his‐
     tory and current directory.

EXAMPLES
     $ skills

     tools(1), about(1), open(1)
